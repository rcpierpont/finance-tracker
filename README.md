This is for the bootdev "first personal project" course.

The finance tracker will be a command line tool written in python. It will be used to conveniently add up recent purchases and categorize them how the user wants.

Over time, this lets the user see where their money is going and come up with saving goals based on areas where they identify they can spend less.

uv recommended because it's what I used and I haven't tested other methods

To update distributable:

uv build --wheel

To install: 

uv pip install <path to whl>
